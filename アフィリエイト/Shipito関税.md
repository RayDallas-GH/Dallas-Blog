<h2 id="anker2">関税・消費税がかかる基準（10,000円ルール）</h2>
<img class="alignnone wp-image-30417 size-large" src="https://ibis-dallas.com/wp-content/uploads/2025/11/u1698565177_abstract_visualization_of_USD_spending_glowing_do_4dd2fa73-ce98-41e1-8352-779e378a98c5_1-1024x574.jpg" alt="アメリカドル" width="728" height="408" />

日本に商品を輸入する際、<strong style="background: linear-gradient(to bottom, rgba(128,255,255,0) 60%, #80ffff 60%);">商品の価格が10,000円を超える場合、関税や消費税が課される</strong>可能性があります。

これを「<strong style="background: linear-gradient(to bottom, rgba(255,255,102,0) 60%, #ffff66 60%);">10,000円ルール</strong>」と呼ぶこともあります。

ただし、<strong style="background: linear-gradient(to bottom, rgba(128,255,255,0) 60%, #80ffff 60%);">商品の種類によっては、10,000円以下でも関税がかかる場合があります</strong>ので注意が必要です。
<h3>10,000円ルールの詳細</h3>
<ul>
 	<li><strong>10,000円以下</strong>：関税・消費税が免除される場合が多い（商品の種類による）</li>
 	<li><strong>10,000円超</strong>：関税・消費税が課される可能性が高い</li>
</ul>
<strong style="background: linear-gradient(to bottom, rgba(255,255,102,0) 60%, #ffff66 60%);">この10,000円の基準は、商品価格と送料の合計（課税価格）に対して適用されます</strong>。

商品価格だけでなく、送料も含めて計算されるため、商品価格が9,000円でも送料が2,000円であれば、合計11,000円となり関税・消費税がかかる可能性があります。
<h3>商品の種類による例外</h3>
以下のような商品は、10,000円以下でも関税がかかる場合があります。
<ul>
 	<li><strong>酒類</strong>：アルコール含有飲料は、金額に関わらず関税がかかります</li>
 	<li><strong>タバコ</strong>：タバコ製品も金額に関わらず関税がかかります</li>
 	<li><strong>香水</strong>：香水は金額に関わらず関税がかかることがあります</li>
</ul>
<h2 id="anker3">関税と消費税の計算方法</h2>
<img class="alignnone wp-image-31011 size-large" src="https://ibis-dallas.com/wp-content/uploads/2025/12/Shipito送料見積もり-1024x268.png" alt="Shipito送料見積もり" width="728" height="191" />

関税と消費税の計算は、以下の順序で行われます。
<ol>
 	<li>課税価格を計算する</li>
 	<li>関税額を計算する</li>
 	<li>消費税額を計算する</li>
</ol>
<h3 id="anker31">課税価格の計算</h3>
<strong style="background: linear-gradient(to bottom, rgba(128,255,255,0) 60%, #80ffff 60%);">課税価格には、商品価格と送料が含まれます</strong>。

課税価格の計算式は以下の通りです。

<strong>課税価格 = 商品価格（円） + 送料（円）</strong>

<strong style="background: linear-gradient(to bottom, rgba(255,255,102,0) 60%, #ffff66 60%);">Shipitoの手数料（受理手数料、同梱手数料など）は課税価格に含まれません</strong>が、送料は含まれるため注意が必要です。
<h3 id="anker32">関税の計算方法</h3>
関税は、以下の計算式で求められます。

<strong>関税額 = 課税価格 × 関税率</strong>

<strong style="background: linear-gradient(to bottom, rgba(128,255,255,0) 60%, #80ffff 60%);">関税率は商品の種類によって異なります</strong>。

主な商品の関税率は以下の通りです。

正確な関税率は、<a href="https://www.customs.go.jp/tariff/index.htm" target="_blank" rel="noopener noreferrer">税関のウェブサイト</a>で確認できます。
<h3 id="anker33">消費税の計算方法</h3>
消費税は、関税を含めた金額に対して10％が課されます。

消費税の計算式は以下の通りです。

<strong>消費税額 = （課税価格 + 関税額）× 10％</strong>

<strong style="background: linear-gradient(to bottom, rgba(128,255,255,0) 60%, #80ffff 60%);">関税額を計算した後に消費税を計算するため、関税額が高い商品ほど消費税額も高くなります</strong>。
<h3 id="anker34">実際の計算例</h3>
具体的な計算例を紹介します。
<h4>例1：iPhone（スマートフォン）を購入した場合</h4>
<ul>
 	<li><strong>商品価格</strong>：$999（約15万円、為替レート1ドル=150円と仮定）</li>
 	<li><strong>送料</strong>：$50（約7,500円）</li>
 	<li><strong>課税価格</strong>：157,500円（150,000円 + 7,500円）</li>
 	<li><strong>関税率</strong>：0%（無税）</li>
 	<li><strong>関税額</strong>：0円</li>
 	<li><strong>消費税額</strong>：15,750円（157,500円 × 10％）</li>
 	<li><strong>合計税金</strong>：15,750円</li>
</ul>
<h4>例2：衣類（関税率10.9%）を購入した場合</h4>
<ul>
 	<li><strong>商品価格</strong>：$100（約15,000円）</li>
 	<li><strong>送料</strong>：$30（約4,500円）</li>
 	<li><strong>課税価格</strong>：19,500円（15,000円 + 4,500円）</li>
 	<li><strong>関税率</strong>：10.9%</li>
 	<li><strong>関税額</strong>：2,125円（19,500円 × 10.9％、端数処理）</li>
 	<li><strong>消費税額</strong>：2,162円（（19,500円 + 2,125円）× 10％、端数処理）</li>
 	<li><strong>合計税金</strong>：4,287円</li>
</ul>
<h4>例3：10,000円以下の商品を購入した場合</h4>
<ul>
 	<li><strong>商品価格</strong>：$50（約7,500円）</li>
 	<li><strong>送料</strong>：$20（約3,000円）</li>
 	<li><strong>課税価格</strong>：10,500円（7,500円 + 3,000円）</li>
</ul>
<strong style="background: linear-gradient(to bottom, rgba(255,255,102,0) 60%, #ffff66 60%);">課税価格が10,000円を超えているため、関税・消費税がかかります</strong>。

商品価格だけでなく、送料も含めて10,000円以下に抑える必要があります。
<h4>実際の支払い実績：iPhone17とアクセサリーを購入した場合</h4>
実際に購入して支払った関税・消費税の実績を紹介します。
<ul>
 	<li><strong>購入商品</strong>：iPhone17を2個、シリコンラバーケース1個、ストラップ1個</li>
 	<li><strong>商品価格</strong>：$1,766（米ドル）</li>
 	<li><strong>Shipito送料（保険込み）</strong>：$61.9（米ドル）</li>
 	<li><strong>課税価格</strong>：商品価格 + 送料 = $1,827.9（米ドル）</li>
 	<li><strong>税関が使用した為替レート</strong>：155.51円/ドル（12月14〜20日週（米ドル）、税関公式週次一覧）</li>
 	<li><strong>課税価格（円換算）</strong>：$1,827.9 × 155.51円 = 約284,260円</li>
 	<li><strong>関税率</strong>：0%（スマートフォンは関税なし）</li>
 	<li><strong>関税額</strong>：0円</li>
 	<li><strong>消費税額</strong>：16,740円（実際の支払い額）</li>
 	<li><strong>合計税金</strong>：16,740円</li>

 	<li><strong>支払い場所</strong>：玄関先</li>
</ul>
<strong style="background: linear-gradient(to bottom, rgba(128,255,255,0) 60%, #80ffff 60%);">スマートフォン（iPhone）は関税が0％（無税）ですが、消費税は課税価格に対して10％がかかります</strong>。

税関が使用した為替レート（155.51円/ドル）で計算すると、課税価格は約284,260円となり、理論上の消費税は約28,426円になりますが、<strong style="background: linear-gradient(to bottom, rgba(255,255,102,0) 60%, #ffff66 60%);">実際の支払い額は16,740円でした</strong>。

<strong style="background: linear-gradient(to bottom, rgba(128,255,255,0) 60%, #80ffff 60%);">計算が微妙に合わない場合もありますが、税関の為替レートや計算方法の参考としてご確認ください</strong>。

実際の関税・消費税の計算は、税関が使用する為替レートや端数処理などの影響で、予想と異なる場合があります。

<strong style="background: linear-gradient(to bottom, rgba(255,255,102,0) 60%, #ffff66 60%);">商品到着時に配送業者から代引きで請求され、玄関先で現金支払いとなりました</strong>。

<h2 id="anker5">オレゴン倉庫利用によるアメリカ消費税免税</h2>
<img class="alignnone wp-image-31012 size-large" src="https://ibis-dallas.com/wp-content/uploads/2025/12/Shipito倉庫-1024x653.jpg" alt="Shipito倉庫" width="728" height="464" />

Shipitoのプレミアムメンバーシップに加入すると、<strong style="background: linear-gradient(to bottom, rgba(128,255,255,0) 60%, #80ffff 60%);">オレゴン州の倉庫を利用でき、消費税（Sales Tax）を回避することが可能</strong>です。

カリフォルニア州は消費税率が高いため、<strong style="background: linear-gradient(to bottom, rgba(255,255,102,0) 60%, #ffff66 60%);">高額商品を購入する場合は、プレミアム会員になってオレゴン州の倉庫を利用する方がお得になることが多い</strong>です。
<h3>アメリカ消費税と日本の関税・消費税の違い</h3>
<ul>
 	<li><strong>アメリカ消費税（Sales Tax）</strong>：アメリカで商品を購入する際にかかる税金（州によって税率が異なる）</li>
 	<li><strong>日本の関税</strong>：日本に商品を輸入する際にかかる税金</li>
 	<li><strong>日本の消費税</strong>：日本に商品を輸入する際にかかる消費税（10％）</li>
</ul>

<h3>Q1. 関税は必ずかかりますか？</h3>
<strong>A. いいえ、商品価格と送料の合計が10,000円以下の場合、関税がかからない場合が多いです。</strong>

ただし、商品の種類によっては、10,000円以下でも関税がかかる場合があります。
<h3>Q2. 予想以上の関税が請求された場合はどうすればいいですか？</h3>
<strong>A. 税関に問い合わせて、関税額が正しいか確認することをおすすめします。</strong>

税関申告書に記載された商品価格や種類が間違っている可能性があります。
<h3>Q3. 関税は誰が支払うのですか？</h3>
<strong>A. 商品を受け取る人が支払います。</strong>

商品到着時に、配送業者から関税・消費税が請求されます。

<strong style="background: linear-gradient(to bottom, rgba(255,255,102,0) 60%, #ffff66 60%);">支払い方法は代引き（現金）となります</strong>。商品が玄関先に届いた際に、現金で支払います。
<h3>Q4. 関税を払わないとどうなりますか？</h3>
<strong>A. 商品を受け取ることができません。</strong>

関税・消費税を支払わないと、商品が税関で留置され、受け取ることができません。
<h3>Q5. アメリカ消費税と日本の関税・消費税の違いは何ですか？</h3>
<strong>A. アメリカ消費税は、アメリカで商品を購入する際にかかる税金です。日本の関税・消費税は、日本に商品を輸入する際にかかる税金です。</strong>

オレゴン州の倉庫を利用することで、アメリカ消費税を回避できますが、日本の関税・消費税は別途かかります。
<h3>Q6. 関税率はどうやって確認できますか？</h3>
<strong>A. 税関のウェブサイトで確認できます。</strong>

<a href="https://www.customs.go.jp/tariff/index.htm" target="_blank" rel="noopener noreferrer">税関のウェブサイト</a>で、商品のHSコード（商品分類コード）を調べて、関税率を確認できます。
<h3>Q7. 送料も課税価格に含まれますか？</h3>
<strong>A. はい、送料も課税価格に含まれます。</strong>

商品価格だけでなく、送料も含めて関税・消費税が計算されます。
<h3>Q8. Shipitoの手数料も課税価格に含まれますか？</h3>
<strong>A. いいえ、Shipitoの手数料（受理手数料、同梱手数料など）は課税価格に含まれません。</strong>

課税価格には、商品価格と送料のみが含まれます。
